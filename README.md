# Trilhas - Logica de Programação
Repositório dedicado aos estudos e atividades da disciplina de Lógica de Programação, contendo exercícios, desafios e afins.
## O Simulador de Finanças Tech
A atividade consiste em desenvolver a lógica inicial de um aplicativo de finanças que recebe o nome e saldo do usuário, calcula uma compra de setup, verifica se ela pode ser realizada, classifica a situação financeira com estruturas condicionais e, ao final, apresenta um menu com opções para visualizar o extrato ou sair do aplicativo. ---> fintech_sim.por
